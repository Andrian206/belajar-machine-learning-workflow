# Machine Learning Workflow Learning Project

## Overview
This repository documents my hands-on learning journey in Dicoding's class **"Belajar Machine Learning untuk Pemula"**, specifically the **"Machine Learning Workflow"** syllabus.

The module is designed to build a solid understanding of the complete machine learning lifecycle—from defining objectives and collecting data to model development, deployment, and monitoring.

## Learning Context
- **Platform:** Dicoding
- **Class:** Belajar Machine Learning untuk Pemula
- **Syllabus Focus:** Machine Learning Workflow
- **Learning Format:** Theory + guided practice + exam

## Module Objective
This module provides end-to-end understanding of machine learning workflow, including:
1. Goal definition and problem framing
2. Data collecting and data loading
3. Data cleaning and transformation
4. Exploratory and explanatory data analysis
5. Data splitting and model building
6. Deployment and monitoring fundamentals
